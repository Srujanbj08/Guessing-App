# PalindromeChecker App

PalindromeChecker App is a console-based Java application designed to validate whether a given string is a palindrome using multiple algorithms and data structures. The project demonstrates fundamental programming concepts, object-oriented programming principles, recursion, collections, and performance analysis through incremental use cases. 

---

## Features

* String-based palindrome checking
* Character array palindrome validation
* Stack and Queue based approaches
* Deque-based optimized palindrome checker
* Linked List palindrome validation
* Recursive palindrome checking
* Case-insensitive and space-ignored validation
* Strategy pattern implementation
* Performance comparison of algorithms
