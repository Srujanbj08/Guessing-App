# GuessingApp

GuessingApp is a console-based Java number guessing game developed using Object-Oriented Programming (OOP) concepts. The application demonstrates game configuration, validation, hint generation, exception handling, file persistence, and controller-based game flow management through modular use cases. 

---

## Features

* Random number guessing game
* Configurable game rules and boundaries
* Guess validation system
* Dynamic hint generation
* Custom exception handling
* File-based score persistence
* Retry and restart game support
* Modular service-based architecture

---

## Concepts Covered

* Object-Oriented Programming
* Encapsulation
* Exception Handling
* File Handling using BufferedWriter
* Random Number Generation

* Input Validation
* Service-Oriented Design
